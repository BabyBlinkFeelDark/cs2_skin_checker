# 🔫 Skin Watcher

**Skin Watcher** — это автоматизированная фоновая служба для мониторинга цен на предметы Steam-инвентаря (CS2, Rust, Dota 2). Проект работает как ваш личный портфельный трейдер: он 24/7 следит за рынком и присылает детальную аналитику в Telegram только тогда, когда наступает идеальный момент для фиксации прибыли или выгодного закупа на просадке.

## 🔥 Главные фишки

- **Умный парсинг (Анти-бан):** Имитация действий реального пользователя с плавающими задержками между запросами для обхода лимитов (HTTP 429) от Steam API.
- **Аналитика временных окон:** Отслеживание динамики цены сразу по нескольким фронтам: за последние 24 часа, за 7 дней и от изначальной "Точки входа" (Baseline price).
- **Учет комиссии Steam:** Бот автоматически высчитывает вашу чистую прибыль с учетом 15% налога торговой площадки.
- **Двойная система уведомлений (Dual Bots):** Строгое разделение логов. Финансовые сигналы (пампы/дампы) летят в одного Telegram-бота, а системные уведомления (старт, краши) — в другого.
- **Неубиваемая очередь (Offline-proof):** При падении интернета или API Telegram ни один алерт не потеряется. Система складывает сообщения в SQLite-очередь, склеивает в пачки и досылает при восстановлении связи.
- **Защита от спама (Cooldowns):** Наличие истории алертов не позволяет боту спамить одним и тем же сигналом (например, рост за 24 часа) чаще, чем раз в сутки.

## 🛠 Технологии

- **Язык:** Python 3.8+
- **База данных:** SQLite (локальное хранилище истории цен, инвентаря и очередей)
- **Библиотеки:** `requests` (сеть), `schedule` (планировщик задач), `win11toast` (нативные уведомления Windows)

## 📦 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/yourusername/skin-watcher.git
   cd skin-watcher
   ```

2. **Установите зависимости:**
   Рекомендуется использовать виртуальное окружение.
   ```bash
   pip install requests schedule win11toast urllib3
   ```

3. **Первый запуск и настройка:**
   ```bash
   python src/main.py
   ```
   При первом запуске скрипт автоматически создаст базу данных `data/app.db` и запросит у вас в консоли необходимые данные для файла `settings.json`:
   - Ваш **SteamID64** (обязательно публичный инвентарь)
   - Токен и Chat ID для **Бизнес-бота** (уведомления о ценах)
   - Токен и Chat ID для **Инфо-бота** (опционально, системные логи)

4. **Фоновая работа:**
   После успешной настройки скрипт перейдет в режим ожидания и будет регулярно обновлять цены, опираясь на интервал из `settings.json` (по умолчанию каждый час).

## ⚙️ Настройка параметров (settings.json)

Файл `data/settings.json` генерируется автоматически, но вы можете отредактировать его вручную:
- `drop_threshold_percent` (по умолчанию `30.0`) — порог падения в процентах для сигнала к покупке.
- `rise_threshold_percent` (по умолчанию `25.0`) — порог роста в процентах для фиксации прибыли.
- `min_difference_dollars` (по умолчанию `0.5`) — защита от микро-колебаний дешевых скинов. Бот проигнорирует рост на 50%, если в деньгах это всего 2 цента.
- `check_interval_hours` (по умолчанию `1`) — частота запуска парсера.
